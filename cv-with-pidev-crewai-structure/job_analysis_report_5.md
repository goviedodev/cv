# 📊 Informe de Análisis de Mercado Laboral

## Resumen Ejecutivo
Se realizó una búsqueda multi-portal (GetOnBoard, LinkedIn, Computrabajo, etc.) enfocada en posiciones para **Senior Spring Boot Java Developer**. Tras analizar múltiples vacantes, se priorizaron aquellas con alta compatibilidad en el ecosistema Java (Spring Boot), Google Cloud y especialmente aquellas que valoran la adopción de herramientas de Inteligencia Artificial (Agentic AI), lo que hace "fit" perfecto con el perfil de Gonzalo Oviedo.

## Análisis de Opciones Encontradas
Se encontraron dos excelentes oportunidades recientes en **GetOnBoard**:

1. **Back-end Developer (Spring Boot) - 42Labs (Santiago, Híbrido)**
   * **Requisitos clave:** Java, Spring Boot, APIs, integraciones y arquitectura escalable.
   * **Compatibilidad:** Alta, encaja con los 15+ años de experiencia backend de Gonzalo.
   * **Contra:** Es rol estrictamente Backend y no aprovecha su experiencia con Agentic AI e infraestructura.

2. **Desarrollador Full-Stack (Java + Angular + IA) - 2Brains (Remoto)**
   * **Requisitos clave:** Java 11-17, Spring Boot, GCP, Testing (JUnit/Mockito).
   * **Requisito diferencial:** **Integración de IA, uso de Copilots, diseño de prompts efectivos y "desarrollo de skills personalizados"**.
   * **Compatibilidad:** Muy Alta. Aunque requiere Angular (Gonzalo usa más Vue/React/Flutter en frontend), su experiencia como "CTO & Co-founder" integrando *Agentic AI con Pi.dev* para "diseñar e implementar el ciclo de vida de agentes (skills, comandos, workflow)" es exactamente lo que la empresa está pidiendo como un diferenciador crítico en esta posición moderna.
