# ¿Por qué estoy interesado en trabajar en Coderslab.io?

## 🎯 Alineación con mi perfil técnico

### Stack Java 21 + Spring Boot (mi expertise principal)
He trabajado con Java y Spring Boot durante más de 15 años, en roles que van desde desarrollador senior hasta líder técnico. Manejo Java 8-21 con soltura y he construido microservicios, APIs RESTful y sistemas transaccionales críticos para banca (Citibank), e-commerce global (Perficient/Caterpillar) y salud (Seven IT). La oferta de Coderslab.io es una oportunidad ideal para aplicar toda esta experiencia en un entorno moderno de pagos digitales.

### Desafío Node.js + NestJS
Aunque mi stack principal es Java, tengo una base sólida en JavaScript/TypeScript y experiencia con frameworks como Vue.js y React. NestJS sigue los mismos patrones que Spring Boot (inyección de dependencias, decoradores, controladores, servicios), por lo que mi curva de aprendizaje será mínima. Me motiva expandir mi polivalencia técnica y dominar ambos ecosistemas backend.

### Microservicios, Clean Architecture y SOLID
Comparto el enfoque de Coderslab.io en arquitectura limpia y principios SOLID. En mi experiencia actual liderando una startup de movilidad, diseñé el sistema completo desde cero aplicando estos principios: microservicios desacoplados, APIs bien definidas, y código mantenible. Es justamente el tipo de desafío que más disfruto.
